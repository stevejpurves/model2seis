Model2Seismic

GANSTERS

What is it?

model2seismic is a system for rapid forward modelling and seismic inversion in one. It uses Generative Adversarial Networks GANs, to map between velocity and the seismic domain in both directions. Once trained on appropriate data, the system understands how to translate between the two domains, can generalise to datasets beyond the training space. It Rocks!

It is Fast

Training is expensive but a trained system can forward model or invert seismic in seconds.

Extensible

So far we have trained sythetic data made with simple convolutional forward modelling. Yet the system performs well on seismic data with unknown sources. The training set can be expanded to include more geology and realistic physics.
